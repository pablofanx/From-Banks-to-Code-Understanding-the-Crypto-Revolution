# From-Banks-to-Code-Understanding-the-Crypto-Revolution
From Banks to Code: Understanding the Crypto Revolution
Decentralized finance, or DeFi, expands on this idea by recreating financial services without intermediaries. Lending, borrowing, trading, and earning yield can all be done through smart contracts. These contracts execute automatically when conditions are met, removing the need for human intervention. While this increases efficiency, it also introduces new risks, such as bugs in code or exploits in poorly designed systems.
